Front end para el formulario utilizado en el proyecto final de la clase de probabilidad y estadística

Una aplicación sencilla de NextJs que permite recuperar datos de una encuesta y, mediante el back en go, guardarlos en un excel que gradúa los resultados de la encuesta de forma automática.

Este repo estaba antes separado en dos, uno para el front y otro para el back para ver los repos originales:

1. front: https://github.com/j-mv137/pia3

2. back: https://github.com/j-mv137/api-pia
